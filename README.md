# 🏬 UmbrellaSales – Smart Inventory System

## 📌 Overview
A backend-based inventory management system designed for handling decoration products that are rented or sold.  
The system enables admin to manage product stock, track inventory usage, handle rentals and sales, and generate billing reports efficiently.

---

## 🛠 Tech Stack
- Java  
- Spring Boot  
- Spring Security  
- MySQL  
- REST APIs  

---

## 👤 Role

### 🔹 Admin
- Manages complete inventory system  
- Tracks product stock and availability  
- Handles product rentals and sales  
- Generates invoices and reports  

---

## 🔥 Core Features
- Product inventory management with real-time stock updates  
- Rental and sales management system  
- Track product allocation (who took, how many, and when)  
- Price management for rental and selling  
- Bill generation for each transaction  
- Report generation for inventory and sales  
- Secure authentication using Spring Security  

---

## 📦 Modules
- Product Management  
- Inventory Tracking  
- Rental Management  
- Sales Management  
- Billing & Invoice Generation  
- Reports & Analytics  

---

## ⚙️ Workflow
1. Admin adds products with quantity and pricing  
2. System maintains current stock availability  
3. Products are given on rent or sold to customers  
4. System records product details (quantity, price, customer info)  
5. Stock is automatically updated after each transaction  
6. Invoice is generated for billing  
7. Reports are generated for tracking inventory and sales
